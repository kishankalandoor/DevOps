Cut Command Reference

Text processing with cut (characters, fields, bytes)

Cut Command is used for text processing , it is used to extract text , column , pattern , numbering ,etc from a file or from a string.

Field Extraction (cut -f)

A field = text between delimiters. Delimiter = separator (space)

here we used cut -f for field extraction ; % cut -f 1-5 demo.txt this willl show 1st to 5th field from each line

Field Selection
apple@Apples-MacBook-Pro DevOps % cut -f 1,4  demo.txt

hello world 
world map 
apple@Apples-MacBook-Pro DevOps % cut -f 1,3  demo.txt
this is a text file 
hi hello 
hello world 
world map

Pipelining the cut command

here we used | operator for pipelining the multiple cut comamnd

Advanced Examples

apple@Apples-MacBook-Pro DevOps % cut-- complement  -d " "  -f 1-2  demo.txt
