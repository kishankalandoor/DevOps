Sed Command Reference

Sed Command in Linux

Stream Editor for filtering and transforming text

sed command is a powerful text processing tool used for substitution, deletion, insertion, and transformation of text in files or streams.

Syntax: s/pattern/replacement/ is the substitution syntax

Substitution (sed 's/')

here we used sed 's/' for substitution ; % sed 's/hi/hello/' demo.txt this will replace first occurence of hi with hello in each line ; remember to close with / otherwise it will give error
