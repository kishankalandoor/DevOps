Pipe Commands

Using pipes to chain commands in Unix/Linux

pipe = multiple command in 1 execution

syntax : command1 | command2 | command3

COMMAND | COMMAND (pipe)
COMMAND > FILE (redirect)
| → command to command
> → output to file
After | → must be a command
After > → must be a file
sudo affects commands, NOT > or |
