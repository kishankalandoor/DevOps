Set Hostname Correctly
Terminal — Set HostName
apple@Apples-MacBook-Pro ~ % sudo scutil --set HostName kishankalandoor
Terminal — Set ComputerName
apple@Apples-MacBook-Pro ~ % sudo scutil --set ComputerName kishankalandoor
Terminal — Set LocalHostName
apple@Apples-MacBook-Pro ~ % sudo scutil --set LocalHostName kishankalandoor
Flush Cache & Verify
Terminal — Flush DNS Cache
apple@Apples-MacBook-Pro ~ % sudo dscacheutil -flushcache
Terminal — Verify New Hostname
apple@Apples-MacBook-Pro ~ % hostname

kishankalandoor
